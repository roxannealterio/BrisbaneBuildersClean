BRISBANE BUILDERS CLEAN - WEBSITE FILES

=====================================================
ADDING YOUR BEFORE / AFTER PHOTOS  (easy - no editing)
=====================================================
Open the "images" folder and replace these 6 files with your own
photos, keeping the EXACT same file names:

   before-1.jpg  +  after-1.jpg     shows as "Kitchen deep clean"
   before-2.jpg  +  after-2.jpg     shows as "Bathroom restored"
   before-3.jpg  +  after-3.jpg     shows as "Carpet steam clean"

- Any photo size works - they auto-crop to fit neatly.
- Best results: take the "before" and "after" from the same spot/angle,
  landscape (wide) photos look best.
- Want different captions (e.g. "Oven" instead of "Kitchen")? Just ask me,
  or edit the wording in index.html.
- The gallery photos work the same way: replace images/gallery-1.jpg ...
  gallery-4.jpg with real job photos when you have them.

After replacing, upload the whole folder to GitHub and hard-refresh.

=====================================================
GOING LIVE
=====================================================
UPLOAD: upload the ENTIRE contents of this folder to your GitHub Pages
repo (all .html, styles.css, script.js, the images folder, sitemap.xml,
robots.txt, manifest.webmanifest, favicons). Re-upload the WHOLE folder
after any change. Keep the CNAME file if you have one.

DOMAIN (brisbanebuildersclean.com.au): verify it is registered to you, point DNS to
GitHub (A records 185.199.108-111 .153 + www CNAME to your github pages
address), then tick "Enforce HTTPS" in Settings > Pages.

FORMS (Formspree, paid plan): the quote wizard and contact form post to
your connected Formspree endpoint; quote photos attach automatically.
Click the confirmation link on the first live submission.
